# Skills

[![skills.sh](https://www.skills.sh/b/williambout/skills)](https://www.skills.sh/williambout/skills)

My collection of [Claude Code skills](https://code.claude.com/docs/en/skills). Each skill lives in its own folder under [`skills/`](skills).

## Skills

| Skill | Description |
| --- | --- |
| [**uidiff**](skills/uidiff) | Before/after UI screenshots — production vs local dev — with a markdown comparison table ready for a GitHub PR. |

## Install

Install everything:

```bash
npx skills add williambout/skills
```

Or a single skill:

```bash
npx skills add williambout/skills --skill uidiff
```

See each skill's README for usage and development setup.
